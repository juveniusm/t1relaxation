# T1 Relaxation

Interactive visualization of **T1 longitudinal relaxation** — fire a 90° RF pulse and watch Mz recover for fat, muscle, and water. Pause to see the corresponding T1-weighted MRI brightness for each tissue, then drag the cursor line to scrub through time.

**Live:** https://juveniusm.github.io/t1relaxation/

## Contents

- `index.html` — the web app (single file, no build step)
- `addin/` — PowerPoint content add-in that embeds the web app in a slide. See [addin/README.md](addin/README.md) for sideload instructions.
